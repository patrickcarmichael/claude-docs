## Before You Start

### Prerequisites

**For all paths, you should have:**
- Basic programming experience (preferably Python)
- Comfort with command line / terminal
- Understanding of basic web concepts
- A code editor (VS Code recommended)

### Setup

1. **Install Python**: Python 3.9+
2. **Install Claude Code**: [Quickstart](../claude-code/getting-started/quickstart.md)
3. **Create API keys**:
   - [Anthropic API Key](../ai-platforms/anthropic/)
   - Pinecone API Key (for RAG path)
   - OpenAI API Key (for embeddings)

4. **Create accounts**:
   - GitHub (for version control)
   - Vercel (for frontend deployment, optional)
   - AWS/GCP (for backend deployment, optional)

## Frequently Asked Questions

**Q: Do I need to take all the paths?**
A: No! Each path stands alone. Choose based on your goals.

**Q: Can I skip the beginner path?**
A: If you already know RAG basics, you can start at intermediate.

**Q: How much will this cost?**
A: Most paths can be done free with generous trial/free tiers. Plan $50-200/month for production.

**Q: How current is this content?**
A: Updated as of November 2025. Check individual docs for latest updates.

**Q: Can I get a certificate?**
A: These are self-paced guides. Create a portfolio project as proof of learning.

**Q: What if I get stuck?**

**Q: How do I build a portfolio?**
A: Build projects alongside each path. Share code on GitHub.

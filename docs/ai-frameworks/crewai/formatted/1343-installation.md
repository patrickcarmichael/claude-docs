## Installation


To use this tool, you need to install the Scrapfly SDK:

```shell  theme={null}
uv add scrapfly-sdk
```

You'll also need to obtain a Scrapfly API key by registering at [scrapfly.io/register](https://www.scrapfly.io/register/).

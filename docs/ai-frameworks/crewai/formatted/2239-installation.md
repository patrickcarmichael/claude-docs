## Installation


To use these tools, you need to install the Patronus package:

```shell  theme={null}
uv add patronus
```

You'll also need to set up your Patronus API key as an environment variable:

```shell  theme={null}
export PATRONUS_API_KEY="your_patronus_api_key"
```

## Ruby

[Ruby library GitHub repo](https://github.com/anthropics/anthropic-sdk-ruby)

**Requirements:** Ruby 3.2.0 or later

**Installation:**

Add to your Gemfile:
```ruby
gem "anthropic", "~> 1.13.0"
```

Then run:
```bash
bundle install
```

## 1. Install Qwen Code

Install Qwen Code globally via npm:
```bash
npm install -g @qwen-code/qwen-code@latest
```

Verify the installation:
```bash
qwen --version
```

**Prerequisites**: Ensure you have Node.js version 20 or higher installed.

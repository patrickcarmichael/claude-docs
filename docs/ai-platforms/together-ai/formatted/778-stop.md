## Stop

Stop a dedicated inference endpoint.

### Usage

```sh
together endpoints stop [OPTIONS] ENDPOINT_ID
```

### Example

```sh
together endpoints stop endpoint-c2a48674-9ec7-45b3-ac30-0f25f2ad9462
```

### Options

| Options  | Description                   |
| -------- | ----------------------------- |
| `--wait` | Wait for the endpoint to stop |

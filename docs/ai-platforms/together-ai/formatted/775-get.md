## Get

Print details for a specific endpoint.

### Usage

```sh
together endpoints get [OPTIONS]
```

### Example

```sh
together endpoints get endpoint-c2a48674-9ec7-45b3-ac30-0f25f2ad9462
```

### Options

| Options  | Description                 |
| -------- | --------------------------- |
| `--json` | Print output in JSON format |

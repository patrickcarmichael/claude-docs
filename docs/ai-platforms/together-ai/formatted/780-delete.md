## Delete

Delete a dedicated inference endpoint.

### Usage

```sh
together endpoints delete [OPTIONS] ENDPOINT_ID
```

### Example

```sh
together endpoints delete endpoint-c2a48674-9ec7-45b3-ac30-0f25f2ad9462
```

## Install Libraries

```shell
  pip install together composio-togetherai
```
```shell
  npm install @composio/core @composio/vercel @ai-sdk/togetherai ai
```

Set your `TOGETHER_API_KEY` environment variable.
```sh
export TOGETHER_API_KEY=***
export COMPOSIO_API_KEY=***
```

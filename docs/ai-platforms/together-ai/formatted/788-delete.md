## Delete

To delete a previously uploaded file:
```sh
together files delete <FILE-ID>
```
Here's a sample output:
```sh
$ together files delete file-d931200a-6b7f-476b-9ae2-8fddd5112308
{
    "id": "file-d931200a-6b7f-476b-9ae2-8fddd5112308",
    "object": "file",
    "deleted": "true"
}
```

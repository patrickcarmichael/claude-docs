## Installing Libraries

```bash
    pip install -U langgraph langchain-together
```
```bash
    pnpm add @langchain/langgraph @langchain/core  @langchain/community
```

Set your Together AI API key:
```bash
  export TOGETHER_API_KEY=***
```

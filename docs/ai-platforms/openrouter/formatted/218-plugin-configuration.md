## Plugin Configuration

Configure web search behavior:

| Parameter     | Type    | Description                               |
| ------------- | ------- | ----------------------------------------- |
| `id`          | string  | **Required.** Must be "web"               |
| `max_results` | integer | Maximum search results to retrieve (1-10) |

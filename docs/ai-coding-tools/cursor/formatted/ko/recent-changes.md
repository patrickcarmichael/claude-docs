# @Recent Changes
Source: https://docs.cursor.com/ko/context/@-symbols/@-recent-changes

최근 수정된 코드를 컨텍스트로 포함

`@Recent Changes` 심볼은 최근 코드 변경을 AI 대화의 컨텍스트로 포함해.

* 변경은 시간순으로 정렬돼
* 가장 최근의 10개 변경을 우선해
* `.cursorignore` 설정을 준수해

# Welcome
Source: https://docs.cursor.com/zh/welcome

了解 Cursor 并快速上手

Cursor 是一款由 AI 驱动的代码编辑器，能理解你的代码库，并通过自然语言帮你更快地写代码。只要描述你想构建或修改的内容，Cursor 就会为你生成代码。
